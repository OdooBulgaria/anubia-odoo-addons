CRM Lead Changelog

This module adds the needed behavior to maintain a history of changes in leads/opportunities.

Installation

To install this module, you need to:

  • This module requires the following modules installed:
    • CRMCustomer Relationship Management.
    • WebOpenERP Web core module.
  • In addition, this module also uses some features provided by web_m2x_options module, this features only work if it is installed.

Configuration

To configure this module, you need to:

  • This module does not have its own configuration.
  • Some features could be affected by the CRM module configuration.

Usage

To use this module, you need to:

  • Only sales managers can use the features in this module.
  • You can access history through the Lead/Opportunity Changelog Changelog.

For further information, please visit:

  • https://www.odoo.com/forum/help-1
