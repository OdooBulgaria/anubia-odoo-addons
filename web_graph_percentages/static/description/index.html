Web Graph Percentages

It allows to calculate percentages in pivot tables

Installation

To install this module, you need to:

  • To install this module must have web_graph in the Odoo addons directory.
  • Module web_graph will be installed together with this module.

Configuration

This module does not need to be configured, once it has been installed the percentages will be available to use in any pivot table.

Usage

To use this module, you shoud:

  • Open any available Graph View.
  • Select the Pivot Table Mode.
  • Select Percentages in the measures menu.

For further information, please visit:

  • https://www.odoo.com/forum/help-1
